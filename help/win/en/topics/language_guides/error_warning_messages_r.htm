Error and Warning Messages

Errors and warning messages are both output in the console (or the Command Window in GUI mode) of OpenMatrix.

Errors output 'Error:' followed by the error message, and warning messages output 'Warning:' followed by the warning message.

Try/Catch typically uses error and/or warning messages.

  • Errors
    There are two types of errors: parse-time, and run-time.
  • Try Catch
    See the Try/Catch description in the Flow of Control and Looping section.
  • Warning Messages
    Warning messages are similar to error messages in that they are output into the console or command window.
