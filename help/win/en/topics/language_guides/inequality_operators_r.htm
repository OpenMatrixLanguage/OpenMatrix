Inequality Operators

Inequality operators include greater than, less than, or equal to, for example.

  • < (less than)
  • <= (less than or equal to)
  • >(greater than)
  • >= (greater than or equal to)
  • == (equal to)
  • ~= (not equal to)
  • = (assignment)

Example

3 < 9
[4 7 2] == [4 0 2]

More examples are located in the Inequality Operators section of the Operator Description (tables) topic.
