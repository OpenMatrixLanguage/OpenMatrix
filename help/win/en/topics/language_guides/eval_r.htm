Eval

Eval is a special function that accepts a string that is itself an OML script. The function evaluates the script and returns the appropriate value.

The string itself can include any valid OML statements.
A=eval('1+2*3')
The value of A after this statement is 7.
Any variables that are visible to the caller of the eval function are also visible inside the string passed to eval.
b=4
a=eval('2*b+1')
The value of A after this statement is 9.
The eval function can be used to create or modify variables in the current scope.
eval('a=4');
a => 4
