Workspace and Scoping Rules

There are two types of workspaces: Local and Global.

Local workspaces are workspaces in which variables are only defined in the local workspace (for example, in a function). Global workspaces allow variables to be defined everywhere. Persistent variables are local, but allow their value to be retained between function calls. For more on local, global and persistent, see global variables, local variables, and/or persistent variables.

  • Persistent
    The persistent keyword is used to declare a variable as persistent. A persistent variable is local to a function (like a normal variable), but its value is retained between function calls. It is initialized to an empty matrix.
