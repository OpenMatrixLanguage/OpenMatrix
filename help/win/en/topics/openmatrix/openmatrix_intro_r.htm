Introduction to OpenMatrix

OpenMatrix Language (OML) is a high level, matrix-based numerical computing language for all types of math, including solving matrix analysis, differential equations to performing signal analysis and control design.

OML is open source to encourage interested scientists and engineers to expand the language, add toolboxes, and employ it for their math modeling and simulation tasks.

The following topics are included:
  • Getting started with OpenMatrix.
  • Using OpenMatrix with external tools, such as ATOM and GNUPLOT.
  • A description of OML.
  • A reference guide describing the functions available in OML.

OpenMatrix is available as source code on GitHub. Click here to find the source code, the instructions to build OpenMatrix, and the required third parties.

OpenMatrix is also available as pre-compiled binaries for the supported plaforms (Windows and Linux).
