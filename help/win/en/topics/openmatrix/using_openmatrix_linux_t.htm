Using OpenMatrix on Linux

  1. After installing OpenMatrix on Linux, invoke OpenMatrix in Console mode using the shell script (run_omlconsole.sh), located in the scripts/ installation folder.
  2. The run_omlconsole.sh file might need to be adapted to specific paths on your machine. Specifically, OpenMatrix offers a bridge between OML and Python. This bridge is disabled by default.
  • Enable the Bridge between OML and Python on Linux
