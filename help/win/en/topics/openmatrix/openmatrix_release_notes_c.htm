Welcome to OpenMatrix

OpenMatrix is an open source project. To learn more about the OpenMatrix project, please visit https://www.openmatrix.org/.

OpenMatrix 1.0.4 includes the following new features and enhancements.

Highlights

  • OML performance improvements.
  • Support for sparse matrices.
  • Improved class support.
  • New OML functions.

OML Improvements

Class Support
Class support is extended and exposed in OML.
Sparse Support
Support for sparse matrices is added to OML, with arithmetic operations, creation, and extraction.

OML language performance enhancements when executing large scripts (parsing).

The following functions have been added to OpenMatrix 1.0.4:
  • accumarray
  • bi2de, de2bi
  • deconv
  • dlmread
  • isrow, iscolumn
  • mscohere
  • rat
  • shiftdim
  • tfestimate
  • vec2mat
The following functions have been improved or extended in OpenMatrix 1.0.4:
  • fprintf
  • rmdir
  • textread
  • textscan

Support for regexp on Linux is improved.
