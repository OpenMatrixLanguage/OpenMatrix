Welcome to OpenMatrix

OpenMatrix is an open source project. To learn more about the OpenMatrix project, please visit https://www.openmatrix.org/.

OpenMatrix 1.0.7 includes the following new features and enhancements.

OML Commands and Functions

The following commands have been improved:

  • Support ss(A,B,C,D) with sparse A.
  • Improved performance for eig and matrix inverse operation.
  • Improved performance of multi-dimensional and 2D matrices interaction.
  • Support [A,B;C,D] with sparse A,B,C,D.
  • for loop performance improvement.
  • delete function can delete curves or graphical objects.
