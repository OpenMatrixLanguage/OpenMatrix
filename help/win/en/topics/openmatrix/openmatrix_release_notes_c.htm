Welcome to OpenMatrix

OpenMatrix is an open source project. To learn more about the OpenMatrix project, please visit https://www.openmatrix.org/.

OpenMatrix 1.0.11 includes the following new features and enhancements.

OML Commands and Functions

The following functions have been added to OpenMatrix 1.0.11:

Data Structure Functions
  • arrayfun
  • structfun
Signal Processing Functions
  • chirp
  • diric
  • gauspuls
  • sawtooth
  • square
  • rectpuls
  • tripuls
Time Functions
  • datestr
  • datetick
  • datevec
  • weekday

Plotting Functions

  • legend
  • colorbar
  • findall
  • stem3
  • quiver
  • Axis properties:
    • xtick, ytick, and ztick
    • xticklabel and yticklabel
    • xminortick, yminortick, and zminortick
    • xaxislocation and yaxislocation

CoreMinimal Interpreter Functions

  • clearvars
  • isbanded
  • isbatch
  • isfloat

Resolved Issues

  • saveas command was not working properly.
  • Properties for text could not be set in the text command.
  • Options were missing from the axis command.
  • bar command did not accept category names.
  • The values of properties were not validated.
  • Legend location was not supported.
  • The drawnow command has been added to OpenMatrix.
  • Some plot commands in Compose were not supported.
  • There was a performance issue when creating subplots.
  • There was a wrong plot for the contour3 example.
