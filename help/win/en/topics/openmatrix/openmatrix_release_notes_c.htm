Welcome to OpenMatrix

OpenMatrix is an open source project. To learn more about the OpenMatrix project, please visit https://www.openmatrix.org/.

OpenMatrix 1.0.12 includes the following new features and enhancements.

OML Commands and Functions

The following functions have been added to OpenMatrix 1.0.12:

Signal Processing Functions
  • resample
  • stft and stft
  • spectrogram
Plotting Image Functions
  • movie
  • getframe
Notebook Functions
  • fanplot
  • patch
  • pcolor
  • polor

Resolved Issues

  • Issue with variable definition when line break + whitespace is used.
  • Crash with 3D empty matrix using max function.
  • Issue with NaN in max() and min().
  • csvwrite doesn't support very large matrices.
  • Issue on rounding choices in linspace.
  • Cannot compare sparse logical matrices.
  • Run command does not work if a given filename is in a path with a directory with utf8 characters.
  • Find is disabled when break point is hit.
  • Right-click response is slow on some plots: area, contour, surf.
  • restoredefaultpath did not restore the path to the same path as on startup.
  • de2bi issue when base is larger than 10.
  • de2bi cannot handle a large number.
  • de2bi should raise an error when input argument is a float number.
  • normrnd (ones (2,1), 2) returns an error.
  • CLONE - parsing issue in a=2;;b=3;;
  • Horizontal scroll bar issue in editor.
  • Copy full path should use backward slash instead of forward slash.
  • Crash with iso6487(nan,nan,60).
