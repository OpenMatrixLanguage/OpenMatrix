Plotting with OpenMatrix

OpenMatrix is a console-mode application and does not include any plotting capabilities.

However, OpenMatrix has a connection to GNUPLOT, a popular and powerful plotting environment. If GNUPLOT is installed and detected by OpenMatrix,all plot commands will be processed by GNUPLOT, otherwise the plot commands are ignored. The connection with GNUPLOT works both for the plain console mode and the ATOM mode.

You must install GNUPLOT and have the GNUPLOT binary folder (C:\Program Files\gnuplot\bin) in the PATH.

The detection is automatic when OpenMatrix starts. Upon successful detection, a message is displayed:
checking gnuplot...
gnuplot 5.2 patchlevel 2
