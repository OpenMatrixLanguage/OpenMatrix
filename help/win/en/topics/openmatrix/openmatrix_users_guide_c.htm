User's Guide

  • Starting OpenMatrix
    Installers can be found on the OpenMatrix website and include pre-compiled versions of the current OpenMatrix release.
  • Using OpenMatrix with ATOM
    ATOM is a powerful text editor that supports customization and external packages.
  • Plotting with OpenMatrix
    OpenMatrix is a console-mode application and does not include any plotting capabilities.
  • Creating Toolboxes for OpenMatrix
    Additional toolboxes can be written in the C/C++ language and interfaced with OML.
