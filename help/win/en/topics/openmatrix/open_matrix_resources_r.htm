OpenMatrix Resources

OpenMatrix is an open source project. In addition to this user's guide, additional information can be found in the resources provided by the OpenMatrix community.

The main resource is the OpenMatrix website (https://www.openmatrix.org/). It contains the latest information, documentation, material, and links to the different tools used by the community:
  • Github https://github.com/OpenMatrixLanguage/OpenMatrix
  • Discourse https://community.openmatrix.org/

Developers and contributors may also be interested in links to the OpenConfluence and OpenJira pages found on the main website.
