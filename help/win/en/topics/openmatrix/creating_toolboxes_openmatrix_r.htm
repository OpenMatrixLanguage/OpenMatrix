Creating Toolboxes for OpenMatrix

Additional toolboxes can be written in the C/C++ language and interfaced with OML.

The interface and the methods exposed are described and illustrated through examples. Provided the toolbox is created as described below, the addition of the toolbox to OML is simply performed by this command: addtoolbox('toolboxname.dll').

  • Writing an OML Toolbox Using the Binary-compatible Interface
    Objects in this interface are size-invariant, which is a benefit to using this interface.
  • Methods for OML Toolboxes
    This section describes the methods exposed in OML to interface new toolboxes.
  • OML Toolbox Method Examples
    Toolbox methods include cell array, matrix, and struct.
