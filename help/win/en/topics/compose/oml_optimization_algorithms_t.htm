OpenMatrix-4030: Optimization Algorithms in OML

Altair Compose OML language provides optimization solvers for function minimization and zero finding. The solvers include fminunc for unconstrained minimization, fmincon and ga for constrained minimization, and fsolve for solving systems of nonlinear equations.

This tutorial demonstrates each solver by finding the minimum distance between two non-intersecting ellipsoids defined as follows:

(x-1)2/25 + (y-2)2/16 + (z-3)2/9 = 1
(x-12)2/49 +(y-13)2/81 + (z-14)2/36 = 1
