winter

Create winter colormap.

Syntax

winter()

winter(n)

cmap = winter(...)

Inputs

n
Length of the colormap (default: 64).
Type: integer
Dimension: scalar

Outputs

cmap
Colormap in matrix format with size n x 3 (red, green and blue).
Type: double
Dimension: matrix

Example

Simple colormap using winter: 
x=[0:0.1:2*pi];
y=x;
z=sin(x')*cos(y);

contour(z);
cmap = winter(256);
colormap(cmap);
