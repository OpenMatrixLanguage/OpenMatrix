close

Close the current, or a specific, or all figure window.

Syntax

close()

close(h)

close('all')

Inputs

h
Handle of a figure/uicontrol element/waitbar that will be closed.
Type: integer
Dimension: scalar

Outputs

h
Handle of the axes which get cleared.

Example

Simple close example.

close(1)
