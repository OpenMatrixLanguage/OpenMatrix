isaxes

Returns true if x is an axis handle; returns false if otherwise.

Syntax

isaxes(h)

Inputs

h
Handle.
Type: double
Dimension: scalar

Outputs

R
Numerical equivalent true/false.

Example

Simple isaxes example.

isaxes('x')
ans = 0
