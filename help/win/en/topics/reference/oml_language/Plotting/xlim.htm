xlim

Set and query range of x axis.

Syntax

xlim()

xlim([X_lo X_hi])

xlim(hAxis, ...])

Inputs

X_lo, X_hi
Lowest and highest x ranges.
Type: double | integer
Dimension: scalar
hAxes
Axis handle.
Type: double
Dimension: scalar

Example

Simple xlim example:

clf;
xlim([2 8])
xlim()
ans = [Matrix] 1 x 2
2  8
