toascii

Converts every character in the string, s, to its ASCII equivalent.

Syntax

R = toascii(s)

Inputs

s
Type: string

Outputs

R
Type: matrix

Example

R = toascii('This is a test.')

R = [Matrix] 1 x 15
84  104  105  115  32  105  115  32  97  32  116  101  115  116  46
