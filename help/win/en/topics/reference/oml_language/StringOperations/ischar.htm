ischar

Returns true if the input, s, is a string and false otherwise.

Syntax

R = ischar(s)

Inputs

s
Type: double | integer | char | string | logical | cell | struct | mat

Outputs

R
Resulting output, true if s is a string.
Type: logical

Example

String input:

R = ischar('This is a string.')
R = 1
