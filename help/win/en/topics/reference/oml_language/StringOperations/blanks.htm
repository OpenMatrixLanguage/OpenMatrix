blanks

Returns a string which consists of n spaces.

Syntax

R = blanks(n)

Inputs

n
Number of spaces in the resulting output string.
Type: Finite, nonnegative integer

Outputs

R
Type: string

Example

Create a string with ten spaces:

R = blanks(10)
R = 
size(R)
ans = [Matrix] 1 x 2
       1 10

Create a string including some spaces:

S = ['A',blanks(5),'B'];
S =A     B
