restoredefaultpath

Resets the current search path for functions to what it was at application startup.

Syntax

restoredefaultpath()

Example

Add a directory to the search path and restore it to the default:

addpath('functions')
path
Current search path:
functions
[Installation Path]/hwx/scripts/oml

restoredefaultpath
path

Current search path:
[Installation Path]/hwx/scripts/oml
