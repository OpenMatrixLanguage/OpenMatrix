mislocked

Returns true if the named function f is locked.

Syntax

mislocked(f)

Inputs

f
Function name.
Type: char | string
Dimension: string

Outputs

R
Numerical equivalent true/false.

Example

Simple mislocked example.

function f
disp('called')
mlock()
end
mislocked('f')
f()
mislocked('f')
clear('f')
f()
munlock('f')
mislocked('f')
clear('f')
ans = 0
called
ans = 1
called
ans = 0
