horzcat

Returns the horizontal concatenation of x1,x2,...,xN along the 2nd dimension.

Syntax

horzcat(x1,x2,....,xN)

Inputs

x1,x2,...,xN
The elements to be concatenated.
Type: double | integer | char | string | logical | struct | cell
Dimension: scalar | string | vector | matrix

Example

Simple horzcat example:

horzcat([1 2 3 4],[5 6 7 8])
ans = [Matrix] 1 x 8
1  2  3  4  5  6  7  8
