gt

Greater than comparison, equivalent to the > operator.

Syntax

gt(x,y)

Inputs

x,y
Anything that can be logically tested.
Type: double | char | logical | struct | cell | integer
Dimension: scalar | vector | matrix

Outputs

R
Numerical representation of true or false (1 or 0).
Type: logical

Examples

Scalar gt example:
gt(2,1)
R = 1
Matrix gt example:
gt([2, 1; 3, 4], [1, 2; 3 4])
ans = [Matrix] 2 x 2
1  0
0  0
String gt example:
gt('test','year')
R = [Matrix] 1 x 4
0  0  1  1

Comments

If two strings are compared then they must have the same length.
