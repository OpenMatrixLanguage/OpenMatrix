arg

Returns the argument (angle) of each element of z in radians. The arguement is defined as atan2(imag(z),real(z)).

Syntax

R = arg(z)

Inputs

z
Any valid scalar | vector | matrix
Type: double | integer
Dimension: scalar | vector | matrix

Examples

Simple arg example:

R = arg(3 + 4i)
R = 0.92730
Simple arg example:
a = eye(2) + eye(2)*i;
R = arg(a)
R = [0.78540 0; 0 0.78540]
