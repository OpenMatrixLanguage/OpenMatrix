normalize

Normalize vectors.

Syntax

R = normalize(x)

Inputs

x
The vector or matrix to normalize. A matrix is normalize by row.
Dimension: vector | matrix

Outputs

R
The normalized result.

Example

normalize([4,3,6,7,8.1])
R = [Matrix] 1 x 5
0.30185 0.22638 0.45277 0.52823 0.61124
