continue

Skips statements after continue and executes the next iteration of the loop.

Syntax

continue

Example

for i = 1:10
    if mod(i, 3) == 0
      printf('Skips [continue] iteration: %d\n', i)
      continue
    end
printf('Execute iteration:          %d\n', i)
end
