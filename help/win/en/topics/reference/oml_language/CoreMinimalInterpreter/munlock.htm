munlock

Unlocks the named function f. If no input is given then munlock unlocks the current function.

Syntax

munlock('f')

Inputs

f
Function name.
Type: char | string
Dimension: string

Example

Simple munlock example.

function f
disp('called')
mlock()
end
mislocked('f')
f()
mislocked('f')
clear('f')
f()
munlock('f')
mislocked('f')
ans = 0
called
ans = 1
called
ans = 0
