rdivide

Perform element-wise matrix right division, equivalent to the ./ operator.

Syntax

rdivide(x,y)

Inputs

x
The numerator.
Dimension: scalar | vector | matrix
y
The denominator.
Dimension: scalar | vector | matrix

Outputs

R
The quotient result.

Examples

Matrix division by a scalar:
R = rdivide([4 6; 2 7], 4)
R = [Matrix] 2 x 2
1.00000  1.50000
0.50000  1.75000
Matrix element-wise division:
R = rdivide([1 2 5 3],[7 4 2 9])
R = [Matrix] 1 x 4
0.14286  0.50000  2.50000  0.33333
