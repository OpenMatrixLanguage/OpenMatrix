Calculus Commands

  • cumtrapz
    Cumulative numerical integration of discrete data using the trapezoid rule.
  • quad
    Numerical integration (quadrature).
  • quadv
    Numerical integration using adaptive Simpson's rule.
  • trapz
    Numerical integration of discrete data using the trapezoid rule.
