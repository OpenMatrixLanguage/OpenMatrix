cond

Compute the condition number of a matrix.

Syntax

R = cond(x)

Inputs

x
The matrix whose condition number is computed.
Dimension: matrix

Outputs

R
The condition number.
Type: scalar

Example

R = cond([1,2,3;4,5,6;7,8,10])
R = 88.4482799
