sin

Returns the sine for each element of the argument (in radians).

Syntax

R = sin(a)

Inputs

a
The angle in radians.
Dimension: scalar | vector | matrix

Outputs

R
scalar | vector | matrix

Example

Scalar input:
R = sin(pi/3)
R = 0.866025404
Matrix input:
R = sin([pi/6, -pi/4; -pi/3, pi/2])
R = [Matrix] 2 x 2
  0.50000  -0.70711
 -0.86603   1.00000
