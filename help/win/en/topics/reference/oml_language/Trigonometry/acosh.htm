acosh

Returns the inverse hyperbolic cosine (in radians) of a.

Syntax

R = acosh(a)

Inputs

a
Any valid scalar | vector | matrix
Type: double | integer
Dimension: scalar | vector | matrix

Outputs

R
scalar | vector | matrix

Examples

Simple scalar input:
R = acosh(-2)
R = 1.31696 + 3.14159i
Simple matrix input:
R = acosh([2,3,4])
R = [ 1.31696 1.76275 2.06344 ]
Complex matrix input:
R = acosh([i+1,-2-i, 6i, 1])
R = [ 1.06128 + 0.904557i, 1.46935-2.63424i, 2.49178 + 1.5708i, 0]
