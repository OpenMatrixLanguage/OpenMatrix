sinh

Returns the hyperbolic sine of a (input in radians).

Syntax

sinh(a)

Inputs

a
Any valid scalar | vector | matrix in radians.
Type: double | integer
Dimension: scalar | vector | matrix

Outputs

R
scalar | vector | matrix

Examples

Simple scalar input:
R = sinh(-0.5)
R = -0.521095
Simple matrix input:
R = sinh([-3,.6])
R = [ -10.0179 0.636654
Complex matrix input:
R = sinh([i+2,3i-1;i,0.3])
R = [ 1.9596 + 3.16578i, 1.16344 + 0.21776i; 0.841471i, 0.30452]
