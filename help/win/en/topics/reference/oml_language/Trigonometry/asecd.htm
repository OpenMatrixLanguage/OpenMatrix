asecd

Returns the inverse secant (in degrees) for each element of the argument.

Syntax

R = asecd(x)

Inputs

x
Dimension: scalar | vector | matrix

Outputs

R
The angle result (in degrees).

Examples

Scalar input:
R = asecd(2/sqrt(3))
R = 30
Matrix input:
R = asecd([1, 2/sqrt(3); 2, sqrt(2)])
R = [Matrix] 2 x 2
  0.00000  30.00000
  60.00000  45.00000
