Trigonometry Commands

  • acos
    Returns the inverse cosine (in radians) of a.
  • acosd
    Returns the inverse cosine (in degrees) for each element of x.
  • acosh
    Returns the inverse hyperbolic cosine (in radians) of a.
  • asin
    Returns the inverse sine (in radians) of a
  • asind
    Returns the inverse sine in degrees of a.
  • asinh
    Returns the inverse hyperbolic sine (in radians) of a.
  • atan
    Returns the inverse tangent (in radians) of a.
  • atan2
    Returns the inverse tangent in radians for each element of x and y.
  • atanh
    Returns the inverse hyperbolic tangent of a.
  • cos
    Returns the cosine of a (input in radians).
  • cosd
    Returns the cosine of a (input in degrees).
  • cosh
    Returns the hyperbolic cosine of a (input in radians).
  • deg2rad
    Converts degrees to radian values.
  • hypot
    Returns the element-wise square root of the sum of the squares of x and y.
  • rad2deg
    Converts radian values to degrees.
  • sin
    Returns the sin of a (in radians).
  • sind
    Returns the sine of a (in degrees).
  • sinh
    Returns the hyperbolic sine of a (input in radians).
  • tan
    Returns the tangent of a (in radians).
  • tanh
    Returns hyperbolic tangent of a.
