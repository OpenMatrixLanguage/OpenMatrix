secd

Returns the secant for each element of the argument (in degrees).

Syntax

R = secd(a)

Inputs

a
The angle in degrees.
Dimension: scalar | vector | matrix

Outputs

R
scalar | vector | matrix

Examples

Scalar input:
R = secd(60)
R = 2
Matrix input:
R = secd([60, -45; -30, 90])
R = [Matrix] 2 x 2
  2.00000  1.41421
  1.15470      Inf
