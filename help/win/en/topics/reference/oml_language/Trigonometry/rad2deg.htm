rad2deg

Converts radian values to degrees.

Syntax

d = rad2deg(r)

Inputs

r
Any valid scalar | vector | matrix in radians.
Type: double
Dimension: scalar | vector | matrix

Outputs

d
The degree value.
scalar | vector | matrix

Examples

Scalar input:
d = rad2deg(pi/6)
d = 30.00000000
Matrix input:
d = rad2deg([pi/6, pi/3; 0, pi])
d = [Matrix] 2 x 2
30.00000000   60.00000000
 0.00000000  180.00000000
