ismember

Returns a logical matrix with the same shape as x where bool(i,j) is 1 (true) if x(i,j) is equal to s.

Syntax

ismember(x,s)

ismember(x,s,'rows')

[bool, index] = ismember(...)

Inputs

x, s
Type: double | integer | char | string | logical | struct | cell
Dimension: scalar | string | vector | matrix
'rows'
Specifies to search for common rows as opposed to common elements.
Type: double | integer | char | string | logical | struct | cell
Dimension: scalar | string | vector | matrix

Outputs

bool, index
Resulting vectors.

Examples

Simple ismember example:

ismember('abcd','baeg')
ans = [Matrix] 1 x 4
1  1  0  0

Simple ismember example with multiple return:

[bool, index]=ismember('abcd','baeg')
bool = [Matrix] 1 x 4
1  1  0  0
index = [Matrix] 1 x 4
2  1  0  0

Simple ismember example with 'rows' option:

ismember([1 2 3; 1 4 5; 1 2 3],[1 2 3],'rows')
ans = [Matrix] 1 x 3
1  0  1
