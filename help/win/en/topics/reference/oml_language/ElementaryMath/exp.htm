exp

For each element of x, returns e^x.

Syntax

R = exp(x)

Inputs

x
Any valid scalar | vector | matrix.
Type: double | integer | logical | struct | cell
Dimension: scalar | vector | matrix

Outputs

R
Resultant scalar | vector | matrix.

Examples

Basic exp example:
R = exp(1)
R = 2.71828

Basic exp example with matrix input:

R = exp([1,-2,3])
R = [ 2.71828 0.135335 20.0855 ]

Imaginary input 

R = exp(3i)
R = -0.98999250 + 0.14112001i
