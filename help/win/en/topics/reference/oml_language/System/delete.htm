delete

Deletes the file(s), f.

Syntax

delete(f)

delete(f1, f2...)

Inputs

f
Name(s) of files to be deleted. Wild cards such as '*' can be used to delete multiple files in a single command.
Type: string

Example

fname = 'D:/Tests/FileManipulation/CreatedFile.txt'
delete(fname)
