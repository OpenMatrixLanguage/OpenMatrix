OML to Python Bridge Commands

  • evalpythonfile
    Evaluates a Python file.
  • evalpythonscript
    Evaluates a Python script.
  • exporttopython
    Exports the OML variable value to the Python variable.
  • getpythonvar
    Imports the Python variable value to the OML variable.
