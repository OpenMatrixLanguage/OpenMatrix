evalpythonfile

Evaluates a Python file.

Syntax

[status,errorMessage] = evalpythonfile (fileName)

Inputs

fileName
A fully qualified Python file name.
Type: string

Outputs

status
Status of the executed file.
1
success
0
failure
Type: number
errorMessage
Error message indicating the failure.

Example

[status,errormessage] = evalpythonfile('C:/test.py')
