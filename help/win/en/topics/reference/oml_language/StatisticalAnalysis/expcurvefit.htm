expcurvefit

Fits an exponential curve to data using a log transformation to create a linear model.

Syntax

expcurvefit(x,y)

Inputs

x,y
Sample curves
Type: double | integer | char | string | logical | struct | cell
Dimension: scalar | string | vector | matrix

Outputs

R
Resulting curve.

Example

Simple expcurvefit example.

x = [1,2,3,4,5,56,7,9]
y = [2,4,5,6,7,764,5,7]
expcurvefit(x,y)
x = [ 1 2 3 4 5 56 7 9 ]
y = [ 2 4 5 6 7 764 5 7 ]
ans = [ 3.09732068569524 0.0985766135674293 ]
