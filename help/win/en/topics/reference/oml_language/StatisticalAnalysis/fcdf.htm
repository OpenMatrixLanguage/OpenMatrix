fcdf

Compute F distribution cumulative distribution function values.

Syntax

p=fcdf(x,m,n)

Inputs

x
Values of the distribution random variable.
Type: double
Dimension: scalar | vector | matrix
m
Degrees of freedom.
Type: integer
Dimension: scalar | vector | matrix
n
Degrees of freedom.
Type: integer
Dimension: scalar | vector | matrix

Outputs

p
Cumulative probability values.

Examples

Simple fcdf example:
x = 3;
m=30;
n=20;
p=fcdf(x,m,n)
p = 0.99362
Vector fcdf example:
x = [1:1:7];
m=30;
n=20;
p=fcdf(x,m,n)
p = [Matrix] 1 x 7
0.48908 0.94534 0.99362 0.99899 0.99979 0.99995 0.99998 0.99999
